# Assets

This directory contains architecture diagrams, images, and other visual assets referenced in the documentation.

## Contents

| Asset | Description |
|---|---|
| *(diagrams to be added)* | Architecture diagrams exported as PNG/SVG from the Mermaid sources in `architecture/` |

## Adding Assets

Place image files (PNG, SVG) in this directory and reference them from Markdown files using relative paths:

```markdown
![Architecture Diagram](../assets/system-context.png)
```

Mermaid source diagrams live in `architecture/` — rendered versions can be exported and stored here for environments that do not render Mermaid natively.
